== Description ==
<a href="http://Eduk.nl/ysi/">Your social icons</a>, use the icons you want and fit them perfect in you pages / posts

The social icons plugin for professionals: optimized positioning and sizing, lean / light / fast, open to use own icons and url's, share content or pages, shortcodes

Two setting screens: <strong>basics</strong> for non experts. Ready for use defaults, easy settings. Specify selection / order icons<br>
 <strong>more for experts</strong> define url's, specify your iconstrip and custom css.

An iconstrip is 6 icons in png format (tranperancy) horizontal combined in 1 png
Example iconstrips, documentation and a tool to build your own are available at http://Eduk.nl/ysi/

== Installation ==

Plugin Search:

 - In /wp-admin/, go to Plugins > Add New
 - Search for "Your social icons"
 - Look for "Your social icons?
 - Click “Install Now?
 - After installation has completed, click “Activate Plugin?
